# Task 1 – Rule-Based Chatbot

## Objective

Develop a simple rule-based chatbot that responds to user inputs using predefined rules and pattern matching techniques.

## Technologies Used

- Python 3
- datetime (Built-in Python module)

## Features

- Greeting responses
- Basic conversation
- Bot introduction
- Date response
- Time response
- Help command
- Thank you response
- Exit command
- Unknown input handling

## Project Structure

```
Task1_Rule_Based_Chatbot/
│
├── chatbot.py
├── requirements.txt
└── README.md
```

## How to Run

```bash
python chatbot.py
```

## Sample Conversation

```
==================================================
          CodBot AI Chatbot
==================================================
Type 'help' to see available commands.
Type 'exit' to quit.

You: hello
Bot: Hello! Nice to meet you.

You: what is your name
Bot: I'm CodBot, a simple rule-based chatbot.

You: time
Bot: Current time is 09:45:10 PM

You: thanks
Bot: You're welcome!

You: exit
Bot: Goodbye! Have a great day.
```
